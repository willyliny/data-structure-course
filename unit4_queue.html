C++ 實作

用 Array 實作 Circular Queue

Circular Queue 實作

C++
class CircularQueue {
private:
    int arr[1000];
    int front, rear, count, capacity;

public:
    CircularQueue(int cap = 1000) 
        : front(0), rear(-1), count(0), capacity(cap) {}
    
    // Enqueue: O(1)
    void enqueue(int x) {
        if (count == capacity) {
            throw runtime_error("Queue Overflow!");
        }
        rear = (rear + 1) % capacity;  // 環狀繞回
        arr[rear] = x;
        count++;
    }
    
    // Dequeue: O(1)
    int dequeue() {
        if (isEmpty()) {
            throw runtime_error("Queue Underflow!");
        }
        int val = arr[front];
        front = (front + 1) % capacity;  // 環狀繞回
        count--;
        return val;
    }
    
    int peek() {
        if (isEmpty()) throw runtime_error("Queue is empty!");
        return arr[front];
    }
    
    bool isEmpty() { return count == 0; }
    int size() { return count; }
};
💡 關鍵：用 % capacity 實現環狀繞回。當指標超過陣列尾端，自動從頭開始。

使用 STL: std::queue

C++
#include <queue>
using namespace std;

queue<int> q;

q.push(10);       // 加入後端
q.push(20);
q.push(30);

q.front();       // 回傳 10（前端）
q.back();        // 回傳 30（後端）
q.pop();         // 移除前端（無回傳！）

q.empty();       // false
q.size();        // 2
⚠️ 注意！和 Stack 一樣，STL 的 pop() 不回傳值！要先用 front() 取值。

使用 STL: std::deque

C++
#include <deque>

deque<int> dq;

dq.push_front(10);  // 前端加入
dq.push_back(20);   // 後端加入
dq.pop_front();      // 前端移除
dq.pop_back();       // 後端移除

dq[0];               // 支援隨機存取！

答對 0 / 5 題
Q1. Queue 的特性是？
A先進先出 (FIFO)
B後進先出 (LIFO)
C隨機存取
D優先順序處理
Queue 是 FIFO (First In, First Out)，最先進入的元素最先被取出。就像排隊，先來的人先被服務。
Q2. 執行以下操作後，queue.front() 的值是？
enqueue(5) → enqueue(10) → dequeue() → enqueue(15)
A5
B10
C15
DQueue 為空
enqueue(5) enqueue(10) 後 Queue 是 [5,10]（5 在 front）。dequeue() 移除 5，變成 [10]。enqueue(15) 後變成 [10,15]，front 是 10。
Q3. 以下哪個演算法使用 Queue 作為核心資料結構？
ADFS（深度優先搜尋）
BBFS（廣度優先搜尋）
CQuick Sort
DBinary Search
BFS 使用 Queue 實現「一層一層」的遍歷。DFS 使用 Stack（或遞迴）。
Q4. Circular Queue 解決了什麼問題？
A讓 dequeue 變成 O(1)
B支援雙端操作
C避免 Array 實作的假溢位（空間浪費）
D支援優先順序
Linear Queue 用 Array 實作時，dequeue 後的空間無法重複利用。Circular Queue 讓 rear 指標繞回開頭，重複使用空出的位置。
Q5. Deque (Double-Ended Queue) 的特性是？
A只能從前端操作
B只能從後端操作
C前端移除，後端加入
D兩端都可以加入和移除
Deque 是 Double-Ended Queue，兩端都可以 push 和 pop。它結合了 Stack 和 Queue 的特性。
