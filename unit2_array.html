Array 陣列

最基礎的線性資料結構 — 連續記憶體中的有序元素集合

💡核心定義：陣列在記憶體中連續儲存、類型相同的元素。每個元素有從 0 開始的整數索引，可以在 O(1) 時間內直接存取任意位置。

宣告語法

C++
// 靜態陣列宣告
int scores[5];                          // 宣告但未初始化
int scores[5] = {90, 85, 72, 68, 95};   // 宣告並初始化
int scores[] = {90, 85, 72, 68, 95};   // 讓編譯器推斷大小

// 存取（O(1)）
scores[0] = 100;       // 修改
int x = scores[2];   // 讀取 → 72

// 遍歷（O(n)）
for (int i = 0; i < 5; i++)
    std::cout << scores[i] << " ";

為什麼陣列是 Cache-Friendly？

⚡CPU 讀取記憶體時會載入整個 Cache Line（通常 64 bytes）。陣列元素連續排列，一次載入可涵蓋多個元素。Linked List 節點分散，每次存取可能觸發 Cache Miss。

指標與陣列

C++
int arr[5] = {10, 20, 30, 40, 50};
int* p = arr;  // arr 本身就是指向首元素的指標

arr[2] == *(arr + 2) == *(p + 2)  // 三種等價，都是 30

sizeof(arr) / sizeof(arr[0])  // 取得陣列長度 → 5

std::vector

動態陣列 — 解決靜態陣列大小固定的問題

📦vector 是 STL 提供的動態陣列，底層是連續記憶體，容量不足時自動翻倍擴容。保有 O(1) 隨機存取，且不受固定大小限制。

常用語法

C++
#include <vector>
using namespace std;

vector<int> v;              // 空 vector
vector<int> v = {1,2,3};  // 初始化
vector<int> v(5, 0);       // 5 個 0

v.push_back(42);    // 末尾加入（均攤 O(1)）
v.pop_back();       // 末尾移除（O(1)）
v[0]               // 隨機存取（O(1)）
v.size()           // 元素數
v.capacity()       // 已分配容量

動態擴容示意

觀察 size 與 capacity 的變化。當 size == capacity 時觸發 realloc：

size 0
capacity 0
⚡ realloc 發生！
點擊 push_back() 觀察擴容行為

知識測驗

確認你對 Array 的掌握程度

答對 0 / 5 題
Q1. 以下哪個操作的時間複雜度是 O(1)？
A在陣列中間插入一個元素
B用 index 讀取 arr[5]
C在未排序陣列中搜尋某值
D刪除陣列第一個元素
arr[5] 直接計算 base + 5×size，無論陣列多大都只需一步。
Q2. int arr[6] 的 base address 是 0x2000，sizeof(int) = 4。arr[3] 的位址是？
A0x2003
B0x2006
C0x200C
D0x2010
0x2000 + 3×4 = 0x2000 + 12(0xC) = 0x200C。
Q3. vector 的 push_back() 均攤 O(1)，但偶爾會變成 O(n)，原因是？
Apush_back 要先搜尋重複元素
B容量不足時重新分配記憶體並複製所有元素
Cvector 底層是 Linked List
Dpush_back 總是 O(n)
擴容時在 Heap 重新分配更大的連續空間，並複製所有元素（O(n)）。但由於每次擴容翻倍，均攤每次 push_back 仍是 O(1)。
Q4. 以下哪個場景不適合使用陣列？
A儲存 100 名學生成績，需要隨機查詢
B實作靜態查詢表，索引已知
C文字編輯器緩衝區（頻繁在任意位置插入/刪除字元）
D儲存一週七天的溫度紀錄
文字編輯器需要頻繁的任意位置插入/刪除，每次 O(n) 搬移成本，更適合 Linked List（如 Rope）或 Gap Buffer 結構。
Q5. 取得靜態陣列長度的正確方式是？
C++
int arr[] = {1, 2, 3, 4, 5};
Aarr.size()
Bsizeof(arr) / sizeof(arr[0])
Carr.length()
Dstrlen(arr)
sizeof(arr) = 5×4 = 20，sizeof(arr[0]) = 4，20÷4 = 5。.size() / .length() 是 vector/string 方法，靜態陣列沒有。
